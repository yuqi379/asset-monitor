## 📊 数据库表结构

```sql
CREATE TABLE assets (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    url             TEXT NOT NULL,           -- 资产 URL
    web_title       TEXT,                    -- 网页标题
    ip              TEXT,                    -- IP 地址
    port            INTEGER,                  -- 端口
    protocol        TEXT,                    -- 协议 (http/https)
    platform        TEXT NOT NULL,           -- 来源平台: hunter/github
    domain          TEXT,                    -- 域名
    province        TEXT,                    -- 省份
    city            TEXT,                    -- 城市
    company         TEXT,                    -- 公司/单位
    industry        TEXT,                    -- 行业
    matched_fields  TEXT,                    -- 匹配字段
    source_keyword  TEXT,                    -- 来源关键词
    source_page     INTEGER,                 -- 来源页码
    status          TEXT DEFAULT '未检测',    -- 状态: 未检测/正常/异常
    create_time     TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    found_date      TIMESTAMP,               -- 发现时间
    last_seen       TIMESTAMP,               -- 最后可见
    UNIQUE(url, platform)                    -- 去重
);
```

## 🌐 API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/assets` | GET | 获取资产列表, 支持 `keyword/platform/page` 参数 |
| `/api/assets/stats` | GET | 获取统计信息 |
| `/api/export/excel` | GET | 导出 Excel |
| `/api/export/csv` | GET | 导出 CSV |
| `/api/collect/trigger` | POST | 触发采集 |
| `/api/detect/trigger` | POST | 触发检测 |
| `/*` | * | 静态文件 (Vue Router History 模式) |

## 📝 GitHub Token 配置步骤

1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 选择权限: `repo` (全部), `read:user`
4. 设置过期时间 (建议 3-6 个月)
5. 将生成的 token 填入 `config.yaml`
